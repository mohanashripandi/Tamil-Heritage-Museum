#🏛️ Tamil Heritage Museum

An interactive Unity-based virtual museum that showcases the rich cultural heritage of Tamil Nadu through immersive 3D exploration. Users can navigate the museum, discover historical artifacts, and learn about Tamil architecture and traditions in an engaging virtual environment.

# 🚀 Overview

Tamil Heritage Museum is a 3D virtual museum developed using Unity. The project aims to preserve and promote Tamil culture by providing an interactive environment where users can explore ancient artifacts and architectural elements.

Players can freely move around the museum, view heritage collections, and experience Tamil history through a modern digital platform.
